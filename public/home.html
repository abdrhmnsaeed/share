LAN Drop

Works over the internet — devices don’t need the same Wi‑Fi. Create a room on your computer; everyone else opens the join link to upload, download, or copy text. Rooms and files are removed automatically after about 1 hour (no database — storage is temporary files on the server).

Install

Add to home screen for quick access.

On iPhone/iPad: Share → Add to Home Screen.

Create a room (PC)

Opens the host screen with QR and join link. Use the same site URL you deployed (set PUBLIC_URL on the server).

Create a room

To start a room, open this website on a computer and tap Create room. On this phone you can join with a link below.
